# store-locator# Store Locator

## Quick Start

Inside the "config/config.env" file add the following:

- NODE_ENV = development;
- MONGO_URI = ######
- GEOCODER_PROVIDER=mapquest
- GEOCODER_API_KEY = ######

# Install dependencies

npm install

# Serve on localhost:8000

npm run dev (nodemon)
or
npm start
